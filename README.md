## Help

### Getting GPU to work in Debian 12

#### Software
* Debian 12
* Gnome 43.6
* Windowing System X11
* Daz Studio 4.21.0.5
* Bottles 51.9
* nvidia-libs v0.7.11-0-g0eea65a
* nvidia-driver 525.125.06-1~deb12u1

#### Guide
* Install "nvidia-driver".
* Install "Bottles".
* Create a new bottle.
* Install "Daz Studio".
* Install "setup_nvlibs.sh".
